# html-AirDraw
AirDraw is a real-time virtual drawing application that allows users to draw on a digital canvas using only hand gestures captured through a webcam. The system uses computer vision and hand-tracking technology to detect finger movements and convert them into drawing actions without requiring a mouse, stylus, or touchscreen.

The system is designed with two different controls for better accuracy and usability:

-Right Hand + Shift Key → Drawing Mode
 Users can draw on the virtual canvas by moving their right hand while holding the Shift key. The Shift key acts as a drawing trigger, helping prevent accidental strokes and improving drawing precision.

-Left Hand → Animation Control
 The left hand is used independently to control animations and interactive visual effects without requiring any keyboard input.
